Потренируйтесь искать и подключать новые зависимости. Склонируйте проект из github-репозитория.
 Этот проект позволяет кодировать с помощью алгоритма Base64 произвольную строку, введённую
  пользователем. Метод main в классе Encoder содержит следующий код:
final Scanner scanner = new Scanner(System.in);
System.out.print("Введите строку => ");
final String input = scanner.nextLine();
final String encoded = org.apache.commons.codec.binary.Base64.encodeBase64String(
    input.getBytes()
);
System.out.println(encoded);

После этого найдите и подключите зависимость с такими настройками:
groupId: commons-codec,
artifactId: commons-codec,
version: 1.15.
